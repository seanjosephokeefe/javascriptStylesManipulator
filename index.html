Change Background Color

Using JavaScript

document.getElementById("ID OF ELEMENT").style.color = COLOR;

document.getElementById("ID OF ELEMENT").style.backgroundColor = COLOR;

Using JQuery

$("#ID OF ELEMENT").css("background-color", "COLOR");

$("#ID OF ELEMENT").css("color", "COLOR");

You can also daisy-chain the properties with JQuery:

$("#ID OF ELEMENT").css("color" : "COLOR 1", "background-color" : "COLOR 2");
